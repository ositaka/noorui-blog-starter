## ویب پر عربی ٹائپوگرافی کا منظرنامہ

ٹائپوگرافی اچھے ڈیزائن کی بنیاد ہے، اور عربی ویب ڈیولپمنٹ میں، صحیح فونٹ کا انتخاب بہت سے لوگوں کے خیال سے کہیں زیادہ اہم ہے۔ عربی رسم الخط میں منفرد خصوصیات ہیں جو فونٹ کے انتخاب کو چیلنجنگ اور اہم دونوں بناتی ہیں: جڑے ہوئے حروف، سیاق و سباق کی شکلیں، اختیاری زیر و زبر، اور مختلف خطاطی کی روایات۔

لاطینی فونٹس کے برعکس جہاں عام طور پر کوئی بھی قابل مطالعہ ٹائپ فیس کام کرے گا، عربی فونٹس پڑھنے کی آسانی، ثقافتی مناسبت، اور تکنیکی کارکردگی پر نمایاں طور پر اثر انداز ہو سکتے ہیں۔ فونٹ کا غلط انتخاب آپ کے مواد کو پڑھنا مشکل یا آپ کے سامعین کے لیے ثقافتی طور پر نامناسب بنا سکتا ہے۔

خوشخبری؟ عربی ویب فونٹ کا منظرنامہ حالیہ برسوں میں نمایاں طور پر پختہ ہو گیا ہے۔ اب ہمارے پاس بہترین اختیارات ہیں جو روایت اور جدید ڈیزائن کی حساسیت میں توازن رکھتے ہیں۔

<Callout type="info">
عربی فونٹس عام طور پر مساوی لاطینی فونٹس سے 2-3 گنا بڑے ہوتے ہیں کیونکہ حروف کی شکلوں کی پیچیدگی، سیاق و سباق کی شکل سازی، اور لیگیچر کی ضروریات کی وجہ سے۔ کارکردگی کی اصلاح ضروری ہے۔
</Callout>

## عربی فونٹ کی اقسام کو سمجھنا

مخصوص فونٹس میں جانے سے پہلے، آئیے دو اہم انداز کے زمروں کو سمجھیں:

### نسخ کا انداز

**نسخ** (نسخ) جسم کے متن کے لیے استعمال ہونے والا معیاری انداز ہے۔ یہ وہی ہے جو آپ زیادہ تر کتابوں، اخبارات اور ویب سائٹس میں دیکھتے ہیں۔ نسخ کی خصوصیات:

- واضح، قابل مطالعہ حروف کی شکلیں
- افقی متن کا بہاؤ
- اعتدال پسند stroke contrast
- گول curves
- طویل اقتباسات پڑھنے کے لیے موزوں

**بہترین استعمال:** جسم کا متن، مضامین، دستاویزات، رسمی مواد

### کوفی کا انداز

**کوفی** (کوفی) زیادہ ہندسی اور زاویہ دار ہے۔ جدید کوفی فونٹس اکثر استعمال ہوتے ہیں:

- عنوانات اور سرخیوں کے لیے
- لوگو اور برانڈنگ
- مختصر، اثر انگیز متن
- جدید، minimalist ڈیزائن

**بہترین استعمال:** سرخیاں، UI عناصر، لوگو، ڈسپلے ٹیکسٹ

<PullQuote>
"صحیح عربی فونٹ صرف جمالیات کے بارے میں نہیں ہے—یہ آپ کے سامعین کے پڑھنے کے نمونوں اور ثقافتی توقعات کا احترام کرنے کے بارے میں ہے۔"
</PullQuote>

## فونٹ کا موازنہ: اہم امیدوار

آئیے آج دستیاب سب سے مقبول اور قابل اعتماد عربی ویب فونٹس کا جائزہ لیں۔

### 1. Noto Sans Arabic

**قسم:** Sans-serif، کوفی سے متاثر
**ڈیزائنر:** Google Fonts
**وزن:** 9 weights (Thin سے Black تک)

**خصوصیات:**
- تمام زبانوں کی حمایت کے Google کے مشن کا حصہ ("Noto" = "No Tofu")
- بہترین Unicode کوریج
- جدید، صاف جمالیات
- تمام سائز میں اچھی طرح کام کرتا ہے
- لاطینی متن کے لیے Noto Sans کے ساتھ قدرتی طور پر جوڑا بناتا ہے

**فائل کا سائز:** ~150KB (عام وزن، woff2)

**بہترین استعمال:**
- ویب ایپلیکیشنز
- کثیر لسانی انٹرفیس
- جدید، غیر جانبدار ڈیزائن
- جب آپ کو وسیع weight variety کی ضرورت ہو

```css
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+Arabic:wght@100..900&display=swap');

body {
  font-family: 'Noto Sans Arabic', sans-serif;
}
```

<Callout type="success">
Noto Sans Arabic ویب ایپلیکیشنز کے لیے ایک بہترین ڈیفالٹ انتخاب ہے۔ اس کی وسیع کریکٹر سپورٹ اور متعدد weights اسے ورسٹائل اور قابل اعتماد بناتی ہیں۔
</Callout>

### 2. Cairo

**قسم:** Sans-serif، ہندسی
**ڈیزائنر:** محمد جابر
**وزن:** 7 weights (ExtraLight سے Black تک)

**خصوصیات:**
- مصری اور شمالی افریقی منڈیوں میں مقبول
- ہندسی، جدید ظاہری شکل
- عصری احساس کے ساتھ صاف لائنیں
- اوپن سورس اور مفت
- روایتی اور جدید کے درمیان اچھا توازن

**فائل کا سائز:** ~120KB (عام وزن، woff2)

**بہترین استعمال:**
- اسٹارٹ اپ ویب سائٹس
- ٹیک کمپنیاں
- جدید برانڈز
- جب آپ تازہ، عصری نظر چاہتے ہیں

```css
@import url('https://fonts.googleapis.com/css2?family=Cairo:wght@200..1000&display=swap');

.heading {
  font-family: 'Cairo', sans-serif;
  font-weight: 700;
}
```

### 3. IBM Plex Sans Arabic

**قسم:** Sans-serif
**ڈیزائنر:** IBM (Bold Monday studio)
**وزن:** 7 weights (Thin سے Bold تک)

**خصوصیات:**
- کارپوریٹ، پیشہ ورانہ ظاہری شکل
- IBM Plex Sans Latin کے ساتھ جوڑا بنانے کے لیے ڈیزائن کیا گیا
- انٹرپرائز ایپلیکیشنز کے لیے بہترین
- مضبوط برانڈ شناخت
- تکنیکی دستاویزات کی جمالیات

**فائل کا سائز:** ~135KB (عام وزن، woff2)

**بہترین استعمال:**
- انٹرپرائز ڈیش بورڈز
- پیشہ ورانہ خدمات
- تکنیکی دستاویزات
- کارپوریٹ ویب سائٹس

```css
@import url('https://fonts.googleapis.com/css2?family=IBM+Plex+Sans+Arabic:wght@100..700&display=swap');

.corporate-text {
  font-family: 'IBM Plex Sans Arabic', sans-serif;
}
```

### 4. Tajawal

**قسم:** Sans-serif، کوفی انداز
**ڈیزائنر:** Boutros International
**وزن:** 5 weights (ExtraLight سے Black تک)

**خصوصیات:**
- ہلکا اور ویب کے لیے بہتر بنایا گیا
- حریفوں سے چھوٹے فائل سائز
- UI عناصر کے لیے اچھا
- صاف، minimalist ڈیزائن
- خاص طور پر سکرین ڈسپلے کے لیے تخلیق کیا گیا

**فائل کا سائز:** ~85KB (عام وزن، woff2)

**بہترین استعمال:**
- موبائل ایپلیکیشنز
- کارکردگی کے لحاظ سے اہم سائٹس
- UI اجزاء
- جب فائل کا سائز اہم ہو

```css
@import url('https://fonts.googleapis.com/css2?family=Tajawal:wght@200..900&display=swap');

.button {
  font-family: 'Tajawal', sans-serif;
  font-weight: 500;
}
```

### 5. Amiri

**قسم:** Serif، نسخ انداز
**ڈیزائنر:** خالد حسنی
**وزن:** 2 weights (Regular، Bold)

**خصوصیات:**
- روایتی نسخ خطاطی کا انداز
- وسیع زیر و زبر کی معاونت شامل
- کلاسیکی عربی متون کے لیے کامل
- اعلیٰ معیار کے ligatures
- روایتی دھاتی ٹائپ سے متاثر

**فائل کا سائز:** ~185KB (عام وزن، woff2)

**بہترین استعمال:**
- ادبی مواد
- مذہبی متون
- روایتی یا رسمی سیاق و سباق
- جب صداقت اہم ہو
- شاعری اور کلاسیکی ادب

```css
@import url('https://fonts.googleapis.com/css2?family=Amiri:wght@400;700&display=swap');

.traditional-text {
  font-family: 'Amiri', serif;
  line-height: 1.8; /* نسخ کو زیادہ line-height کی ضرورت ہے */
}
```

<Callout type="info">
Amiri میں قرآنی متن کے لیے وسیع معاونت شامل ہے، بشمول تمام زیر و زبر کی علامات اور خصوصی علامات۔ اگر آپ مذہبی مواد دکھا رہے ہیں، تو یہ آپ کا بہترین انتخاب ہے۔
</Callout>

### 6. Readex Pro

**قسم:** Variable font، Sans-serif
**ڈیزائنر:** Thomas Jockin (Google Fonts کے لیے)
**وزن:** Variable (200-700)

**خصوصیات:**
- جدید variable font ٹیکنالوجی
- تمام weights کے لیے ایک فائل
- سکرین پڑھنے کے لیے بہتر بنایا گیا
- عصری ڈیزائن
- پڑھنے کے انٹرفیس کے لیے بہترین

**فائل کا سائز:** ~145KB (variable font، woff2)

**بہترین استعمال:**
- پڑھنے کی ایپس
- ای کتابیں اور طویل مواد
- جب آپ کو متعدد weights کی مؤثر طریقے سے ضرورت ہو
- جدید ویب ایپلیکیشنز

```css
@import url('https://fonts.googleapis.com/css2?family=Readex+Pro:wght@200..700&display=swap');

.reading-interface {
  font-family: 'Readex Pro', sans-serif;
  font-variation-settings: 'wght' 350; /* Variable font کنٹرول */
}
```

## عربی اور لاطینی فونٹس کا جوڑا بنانا

زیادہ تر دو لسانی ویب سائٹس کو عربی اور لاطینی دونوں رسم الخط کی ضرورت ہوتی ہے۔ یہاں انہیں مؤثر طریقے سے جوڑنے کا طریقہ ہے:

### جوڑا بنانے کے اصول

1. **ملتا جلتا x-height:** لاطینی اور عربی فونٹس کے تناسب ملتے جلتے ہونے چاہئیں
2. **میچنگ وزن:** دونوں فونٹس یکساں طور پر بھاری یا ہلکے محسوس ہونے چاہئیں
3. **ہم آہنگ شخصیت:** ہندسی ہندسی کے ساتھ، humanist humanist کے ساتھ
4. **مستقل stroke کی چوڑائی:** ایک جیسی موٹائی بصری ہم آہنگی پیدا کرتی ہے

### تجویز کردہ جوڑے

<WideBox>

```css
/* جوڑا 1: Noto Sans Arabic + Noto Sans */
:root {
  --font-ar: 'Noto Sans Arabic', sans-serif;
  --font-en: 'Noto Sans', sans-serif;
}

/* جوڑا 2: Cairo + Inter */
:root {
  --font-ar: 'Cairo', sans-serif;
  --font-en: 'Inter', sans-serif;
}

/* جوڑا 3: IBM Plex Sans Arabic + IBM Plex Sans */
:root {
  --font-ar: 'IBM Plex Sans Arabic', sans-serif;
  --font-en: 'IBM Plex Sans', sans-serif;
}

/* جوڑا 4: Tajawal + Roboto */
:root {
  --font-ar: 'Tajawal', sans-serif;
  --font-en: 'Roboto', sans-serif;
}

/* جوڑا 5: Amiri + Crimson Pro */
:root {
  --font-ar: 'Amiri', serif;
  --font-en: 'Crimson Pro', serif;
}

/* زبان کی شناخت کے ساتھ استعمال */
body {
  font-family: var(--font-en);
}

:lang(ar) {
  font-family: var(--font-ar);
}
```

</WideBox>

<Callout type="success">
Noto فونٹ فیملی (Noto Sans Arabic + Noto Sans) خاص طور پر تمام رسم الخط میں ایک ساتھ کام کرنے کے لیے ڈیزائن کی گئی ہے۔ اگر آپ یقینی ہم آہنگی چاہتے ہیں تو وہیں سے شروع کریں۔
</Callout>

## کارکردگی کی اصلاح کی حکمت عملی

عربی فونٹس بڑے ہوتے ہیں۔ یہاں انہیں مؤثر طریقے سے لوڈ کرنے کا طریقہ ہے:

### 1. اپنے فونٹس کو subset کریں

صرف وہ کریکٹرز لوڈ کریں جن کی آپ کو ضرورت ہے:

```html
<!-- Google Fonts: text پیرامیٹر بیان کریں -->
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+Arabic:wght@400;700&text=أبتثجحخدذرزسشصضطظعغفقكلمنهويءآإؤئ" rel="stylesheet">
```

یا font subsetting ٹولز استعمال کریں:

```bash
# glyphhanger استعمال کرتے ہوئے
npm install -g glyphhanger

# فونٹ فائل کو subset کریں
glyphhanger --subset=font.ttf --formats=woff2 --css
```

### 2. font-display استعمال کریں

فونٹس کی لوڈنگ کو کنٹرول کریں اور لے آؤٹ shifts سے بچیں:

```css
@font-face {
  font-family: 'Noto Sans Arabic';
  src: url('/fonts/noto-sans-arabic.woff2') format('woff2');
  font-display: swap; /* fallback دکھائیں، لوڈ ہونے پر swap کریں */
  font-weight: 400;
  font-style: normal;
}
```

**font-display اختیارات:**
- `swap` - فوری طور پر fallback دکھائیں، لوڈ ہونے پر swap کریں (تجویز کردہ)
- `block` - مختصر invisible مدت، پھر فونٹ دکھائیں
- `fallback` - بہت مختصر invisible مدت، جلدی لوڈ ہو تو swap کریں، ورنہ fallback کے ساتھ رہیں
- `optional` - براؤزر کو کنکشن کی رفتار کی بنیاد پر فیصلہ کرنے دیں

<Callout type="warning">
عربی فونٹس کے لیے `font-display: block` سے بچیں۔ فائل کے سائز طویل blocking periods کا سبب بن سکتے ہیں، جو خراب صارف تجربہ پیدا کرتے ہیں۔ بجائے `swap` استعمال کریں۔
</Callout>

### 3. اہم فونٹس کو پہلے سے لوڈ کریں

fold کے اوپر استعمال ہونے والے فونٹس کے لیے:

```html
<link rel="preload" href="/fonts/noto-sans-arabic-regular.woff2" as="font" type="font/woff2" crossorigin>
```

### 4. متعدد weights کے لیے Variable فونٹس

اگر آپ کو بہت سے weights کی ضرورت ہے تو variable fonts پر غور کریں:

```css
@font-face {
  font-family: 'Readex Pro';
  src: url('/fonts/readex-pro-variable.woff2') format('woff2-variations');
  font-weight: 200 700; /* ایک فائل میں مکمل رینج */
  font-display: swap;
}

/* کوئی بھی وزن استعمال کریں */
.heading {
  font-weight: 350; /* عین وزن */
}
```

## لائن کی اونچائی اور spacing کے تحفظات

عربی متن کو لاطینی سے مختلف spacing کی ضرورت ہے:

```css
/* لاطینی متن */
.text-latin {
  font-size: 16px;
  line-height: 1.5; /* 24px */
  letter-spacing: 0;
}

/* عربی متن */
.text-arabic {
  font-size: 16px;
  line-height: 1.8; /* 28.8px - زیر و زبر کے لیے زیادہ جگہ */
  letter-spacing: 0; /* عربی میں کبھی letter-spacing شامل نہ کریں! */
}
```

<Callout type="error">
عربی متن پر کبھی `letter-spacing` استعمال نہ کریں۔ یہ حروف کے درمیان کنکشن کو توڑتا ہے اور رسم الخط کی بصری سالمیت کو تباہ کرتا ہے۔
</Callout>

**جدید اسٹارٹ اپ یا ٹیک پروڈکٹ بنا رہے ہیں؟** → Cairo

**موبائل یا کارکردگی کے لیے بہتر بنا رہے ہیں؟** → Tajawal

**روایتی، ادبی، یا مذہبی مواد دکھا رہے ہیں؟** → Amiri

**پڑھنے کا انٹرفیس بنا رہے ہیں؟** → Readex Pro

**متعدد weights کی مؤثر طریقے سے ضرورت ہے؟** → Readex Pro (variable font)

## اہم نکات

1. **فونٹ کا انداز اہم ہے**: جسم کے متن کے لیے نسخ، سرخیوں اور UI کے لیے کوفی منتخب کریں
2. **فائل کا سائز اہم ہے**: عربی فونٹس لاطینی مساویوں سے 2-3x بڑے ہوتے ہیں
3. **جوڑا بنانا اہم ہے**: عربی اور لاطینی فونٹس کے درمیان x-height، وزن، اور شخصیت کو match کریں
4. **لوڈنگ کو بہتر بنائیں**: font-display: swap، subsetting، اور preloading استعمال کریں
5. **لائن کی اونچائی مختلف ہے**: عربی کو لاطینی (1.4-1.6) سے زیادہ line-height (1.6-2.0) کی ضرورت ہے
6. **کبھی letter-space نہ کریں**: عربی ایک جڑا ہوا رسم الخط ہے—spacing اسے توڑتا ہے
7. **Variable fonts bytes بچاتے ہیں**: اگر آپ کو متعدد weights کی ضرورت ہے تو variable fonts پر غور کریں
8. **سیاق و سباق اہم ہے**: روایتی مواد کو روایتی فونٹس کی ضرورت ہے؛ جدید ایپس کو جدید فونٹس کی ضرورت ہے
