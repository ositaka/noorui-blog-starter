## مقدمة

عندما تكون الحروف اللاتينية بجانب بعضها، تبقى أشكالاً منفصلة: "fl" هي ببساطة "f" متبوعة بـ "l". في العربية، الحروف لا تجلس معاً فحسب — بل *تتحول*. نفس الحرف يمكن أن يبدو مختلفاً تماماً حسب موقعه وجيرانه.

هذا النظام من الحروف المتصلة والمشكّلة سياقياً أساسي للخط العربي. للمطورين والمصممين، فهم الوصلات ضروري للعرض الصحيح للنص واختيار الخطوط وتصحيح مشاكل العرض.

## ما هي الوصلات؟

### التعريف

**الوصلة** هي رمز واحد يُنشأ بدمج حرفين أو أكثر. في العربية، الوصلات ليست زخارف طباعية اختيارية — بل مطلوبة للعرض الصحيح.

### الوصلات في الخطوط اللاتينية

الخطوط اللاتينية تستخدم الوصلات اختيارياً لأسباب جمالية:

```
fi → ﬁ  (يمنع خطاف f من الاصطدام بنقطة i)
fl → ﬂ  (اتصال أكثر سلاسة)
ff → ﬀ  (بديل أنيق)
```

هذه زخرفية بحتة. النص قابل للقراءة بدونها.

### الوصلات في الخطوط العربية

الوصلات العربية إلزامية. الحروف تتصل فيزيائياً، وأشكالها تتغير بناءً على الموقع:

```
ب (باء) وحدها: ب
ب في البداية: بـ
ب في الوسط: ـبـ
ب في النهاية: ـب

مجتمعة: كتب (ك-ت-ب) = ك + ت + ب → أشكال مختلفة تماماً
```

## الأشكال الموضعية الأربعة

كل حرف عربي له حتى أربعة أشكال:

| الموقع | الاسم | الوصف | مثال (ب) |
|--------|-------|-------|----------|
| منفصل | مفرد | مستقل | ب |
| ابتدائي | ابتدائي | بداية الكلمة | بـ |
| وسطي | وسطي | وسط الكلمة | ـبـ |
| نهائي | نهائي | نهاية الكلمة | ـب |

### الحروف غير المتصلة

ستة حروف عربية تتصل فقط من اليمين ولها شكلان فقط (منفصل ونهائي):

```
ا (ألف)
د (دال)
ذ (ذال)
ر (راء)
ز (زاي)
و (واو)
```

هذه الحروف تكسر الاتصال، مما يجبر الحرف التالي على الشكل الابتدائي.

## أنواع الوصلات العربية

### الوصلات الإلزامية

هذه يجب أن تُعرض بشكل صحيح ليكون النص مقروءاً:

**١. لا (لام-ألف)**
أشهر وصلة عربية. عندما يلتقي ل (لام) بـ ا (ألف)، يشكلان شكلاً مدمجاً خاصاً:

```
ل + ا → لا
ليس: لـا (خطأ)
```

**٢. الاتصالات الموضعية**
جميع الأشكال السياقية هي تقنياً وصلات — أشكال الحروف تتغير عند الاتصال.

### الوصلات الاختيارية (الزخرفية)

بعض الخطوط تتضمن وصلات إضافية لأسباب جمالية:

```
لله - شكل خاص في كثير من الخطوط
لمـ (تركيبات لام-ميم)
بسم (تركيبات بسم الله)
```

## كيف تعمل الوصلات تقنياً

### يونيكود وتخزين الحروف

النص العربي يُخزن بالترتيب المنطقي (أول حرف يُكتب = أول في الذاكرة)، ليس الترتيب البصري:

```
كلمة: كتاب
يونيكود: U+0643 U+062A U+0627 U+0628
مخزن: ك ت ا ب
معروض: كتاب (الحروف تتصل وتتحول)
```

### تشكيل OpenType

محركات عرض الخطوط تطبق قواعد تشكيل متعددة:

**١. GSUB (استبدال الرموز)**
- يستبدل التسلسلات بوصلات
- يختار الأشكال الموضعية
- يطبق قواعد خاصة باللغة

**٢. GPOS (تموضع الرموز)**
- يضبط التباعد بين الرموز
- يموضع علامات التشكيل
- يضبط التقنين الدقيق

### التحكم بـ CSS

```css
/* الافتراضي: الوصلات مفعلة */
.arabic-text {
  font-feature-settings: "liga" 1, "rlig" 1;
}

/* تعطيل الوصلات الاختيارية */
.no-fancy {
  font-feature-settings: "dlig" 0;
}
```

## التنفيذ العملي

### متطلبات الخط

للعرض العربي الصحيح، الخطوط يجب أن تتضمن:

١. **جميع الأشكال الموضعية** لكل حرف
٢. **وصلات لام-ألف** (كحد أدنى)
٣. **جداول OpenType المناسبة** (GSUB, GPOS)
٤. **تموضع العلامات** للتشكيل

### اختبار دعم الوصلات

```html
<!-- اختبار وصلة لام-ألف -->
<p lang="ar">لا إله إلا الله</p>

<!-- اختبار الأشكال الموضعية -->
<p lang="ar">ببب ككك ممم</p>

<!-- اختبار الحروف غير المتصلة -->
<p lang="ar">دادا رارا وووو</p>
```

## اعتبارات الأداء

### حجم ملفات الخطوط

الخطوط العربية مع دعم وصلات كامل أكبر من خطوط اللاتينية:

```
خط لاتيني فقط: ~20KB
خط عربي (نسخ): ~100-300KB
خط عربي (نستعليق): 1-5MB
```

## النقاط الرئيسية

١. **الوصلات إلزامية**: على عكس اللاتينية، الوصلات العربية مطلوبة للعرض الصحيح للنص.

٢. **الأشكال الموضعية هي وصلات**: الأشكال الأربعة لكل حرف هي تقنياً استبدالات وصلات.

٣. **لام-ألف حرجة**: وصلة لا يجب أن تعمل ليكون النص مقبولاً.

٤. **OpenType يتعامل مع التعقيد**: تقنية الخطوط الحديثة تدير آلاف قواعد التشكيل تلقائياً.

٥. **اختبر بخطوط حقيقية**: تحقق دائماً من العرض العربي بخطوط عربية مناسبة، ليس بدائل عامة.
