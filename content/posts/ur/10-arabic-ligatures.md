## تعارف

عربی رسم الخط لاطینی یا سیریلک سے بنیادی طور پر مختلف ہے۔ جہاں انگریزی میں ہر حرف الگ رہ سکتا ہے، عربی میں حروف ایک دوسرے سے جڑ کر مسلسل بہتے متن کی شکل لیتے ہیں۔ یہ صرف جمالیاتی پسند نہیں — یہ رسم الخط کا بنیادی حصہ ہے۔

یہ جڑنے کا نظام **جوڑ** (ligatures) اور **سیاقی شکلیں** (contextual forms) کہلاتا ہے۔ ان کو سمجھنا ہر اس شخص کے لیے ضروری ہے جو عربی، فارسی، یا اردو کے ساتھ ڈیجیٹل طور پر کام کرتا ہے۔

## سیاقی شکلیں

### ایک حرف، چار شکلیں

عربی کے زیادہ تر حروف کی چار مختلف شکلیں ہوتی ہیں:

| شکل | نام | پوزیشن | مثال (ب) |
|-----|-----|--------|---------|
| معزول | Isolated | اکیلا | ب |
| ابتدائی | Initial | لفظ کے شروع میں | بـ |
| وسطی | Medial | لفظ کے درمیان میں | ـبـ |
| انتہائی | Final | لفظ کے آخر میں | ـب |

### عملی مثال

لفظ "کتاب" (kitab) میں:

```
ک ← ابتدائی شکل (شروع میں)
ت ← وسطی شکل (درمیان میں)
ا ← غیر جوڑنے والا حرف
ب ← انتہائی شکل (آخر میں)

نتیجہ: کتاب
```

### غیر جوڑنے والے حروف

چھ حروف کبھی بائیں طرف نہیں جڑتے:

```
ا (الف)    -    د (دال)    -    ذ (ذال)
ر (را)     -    ز (زے)     -    و (واؤ)
```

یہ حروف آنے کے بعد اگلا حرف نئے سلسلے کا آغاز کرتا ہے۔

## جوڑ (Ligatures)

### لازمی جوڑ

کچھ حروف کے امتزاج کی خاص جڑی ہوئی شکلیں ہیں:

**لام-الف (لا)**
```
ل + ا = لا
```
یہ سب سے مشہور عربی جوڑ ہے اور یونیکوڈ میں الگ کوڈ پوائنٹ رکھتا ہے۔

**اللہ کا جوڑ**
```
ا + ل + ل + ه = ﷲ
```

### اختیاری جوڑ

خطاطی میں سینکڑوں اضافی جوڑ ہیں جو متن کو خوبصورت بناتے ہیں:

```
فی، می، بی، تی، ثی، جی...
```

## ڈیجیٹل نفاذ

### OpenType خصوصیات

جدید فونٹس OpenType خصوصیات کا استعمال کرتے ہیں:

```css
.arabic-text {
  /* سیاقی شکلیں فعال کریں */
  font-feature-settings:
    "calt" 1,  /* سیاقی متبادل */
    "liga" 1,  /* معیاری جوڑ */
    "rlig" 1,  /* لازمی جوڑ */
    "cswh" 1;  /* سیاقی سواش */

  /* یا آسان طریقہ */
  font-variant-ligatures: common-ligatures;
}
```

### خصوصیات کی وضاحت

| خصوصیت | نام | مقصد |
|--------|-----|------|
| calt | Contextual Alternates | سیاقی شکلیں |
| liga | Standard Ligatures | عام جوڑ |
| rlig | Required Ligatures | لازمی جوڑ (لا) |
| dlig | Discretionary Ligatures | اختیاری جوڑ |
| cswh | Contextual Swash | سیاقی زیبائشی |

### JavaScript میں جوڑ

```javascript
// جوڑ کی موجودگی چیک کریں
function hasLigature(text) {
  const ligatures = ['لا', 'ﻻ', 'ﷲ'];
  return ligatures.some(lig => text.includes(lig));
}

// یونیکوڈ نارملائزیشن
const text = "لا";
const normalized = text.normalize('NFC');
// NFC form جوڑ کو ایک یونٹ رکھتا ہے
```

## فونٹ کی پیچیدگی

### گلفس کی تعداد

```
لاطینی فونٹ:        ~100-200 گلفس
عربی نسخ فونٹ:     ~400-800 گلفس
عربی نستعلیق فونٹ: ~20,000+ گلفس
```

### GSUB ٹیبل

OpenType فونٹس میں GSUB (Glyph Substitution) ٹیبل جوڑ اور سیاقی شکلوں کو ہینڈل کرتا ہے:

```
قواعد کی مثال:
1. اگر ب کے بعد ا آئے → بـا استعمال کرو
2. اگر ل کے بعد ا آئے → لا (جوڑ) استعمال کرو
3. اگر حرف لفظ کے آخر میں ہو → انتہائی شکل استعمال کرو
```

## عام مسائل

### مسئلہ 1: جوڑ نہیں بن رہے

```css
/* غلط - جوڑ غیر فعال */
.broken-ligatures {
  font-feature-settings: "liga" 0;
}

/* صحیح - جوڑ فعال */
.working-ligatures {
  font-feature-settings: "liga" 1, "calt" 1;
}
```

### مسئلہ 2: حروف الگ الگ نظر آتے ہیں

```html
<!-- غلط - غیر عربی فونٹ -->
<p style="font-family: Arial;">مرحبا</p>

<!-- صحیح - عربی فونٹ -->
<p style="font-family: 'Noto Naskh Arabic';">مرحبا</p>
```

### مسئلہ 3: لام-الف ٹوٹا ہوا

```javascript
// غلط - الگ حروف
const broken = 'ل' + 'ا'; // کچھ سسٹمز میں الگ نظر آئے

// صحیح - یونیکوڈ جوڑ
const ligature = '\u0644\u0627'; // لا
// یا براہ راست
const direct = 'لا';
```

## CSS بہترین طریقے

### بنیادی سیٹ اپ

```css
/* عربی متن کے لیے بنیادی سیٹ اپ */
.arabic-base {
  font-family: 'Noto Naskh Arabic', 'Amiri', serif;
  direction: rtl;
  text-align: right;

  /* OpenType خصوصیات */
  font-feature-settings:
    "kern" 1,
    "liga" 1,
    "calt" 1,
    "rlig" 1;
}
```

### خطاطی انداز

```css
/* زیادہ زیبائشی جوڑ */
.calligraphic {
  font-feature-settings:
    "liga" 1,
    "dlig" 1,  /* اختیاری جوڑ */
    "cswh" 1,  /* سواش */
    "salt" 1;  /* سٹائلسٹک متبادل */
}
```

### سادہ انداز

```css
/* سادہ، صاف متن */
.plain-text {
  font-feature-settings:
    "liga" 1,
    "calt" 1,
    "dlig" 0;  /* اختیاری جوڑ بند */
}
```

## اہم نکات

۱. **سیاقی شکلیں لازمی ہیں**: عربی حروف کی چار شکلیں ہیں جو پوزیشن کے مطابق استعمال ہوتی ہیں — یہ اختیاری نہیں۔

۲. **لام-الف خاص ہے**: یہ سب سے اہم جوڑ ہے اور ہر فونٹ میں ہونا ضروری ہے۔

۳. **OpenType ضروری ہے**: جدید عربی فونٹس کو OpenType خصوصیات کی ضرورت ہے۔

۴. **فونٹ اہم ہے**: غلط فونٹ استعمال کرنے سے جوڑ ٹوٹ جائیں گے۔

۵. **ٹیسٹنگ ضروری ہے**: مختلف براؤزرز اور آلات پر ٹیسٹ کریں کیونکہ رینڈرنگ مختلف ہو سکتی ہے۔
