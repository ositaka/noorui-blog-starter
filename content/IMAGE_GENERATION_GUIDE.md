# Kitab Image Generation Guide

This document lists all placeholder images that need to be created using ChatGPT/DALL-E to match Kitab's sleek, modern design aesthetic.

## Design Guidelines

**Kitab's Visual Identity:**
- **Style:** Modern, minimalist, professional, clean
- **Color Palette:**
  - Primary: Purple/Violet (#7c3aed, #8b5cf6) and Blue tones
  - Background: Light grays (#f9fafb, #f3f4f6) for light mode
  - Text: Dark grays (#1f2937, #111827) on light, white (#ffffff) on dark
  - Accents: Subtle shadows, soft borders, refined spacing
- **Typography:** Clean sans-serif fonts (Inter, system fonts)
- **Layout:** Card-based, generous white space, subtle depth with shadows
- **Feel:** Professional, educational, technical but approachable

**General Image Specifications:**
- **Format:** JPG for photos/complex images, PNG for screenshots/diagrams
- **Resolution:** 1200px width minimum for featured images, 800px for inline images
- **Quality:** High-definition, crisp, professional
- **Consistency:** All images should feel like part of the same visual system
