## تعارف

عربی رسم الخط کی دنیا میں دو نام سب سے زیادہ ابھر کر سامنے آتے ہیں: **نسخ** اور **نستعلیق**۔ یہ دونوں انداز ایک ہی بنیادی حروف تہجی استعمال کرتے ہیں، لیکن ان کی شکل، استعمال، اور ثقافتی اہمیت بہت مختلف ہے۔

ڈیزائنرز اور ڈویلپرز کے لیے ان دونوں اندازوں کو سمجھنا ضروری ہے — خاص طور پر جب عربی، فارسی، یا اردو کے لیے ڈیجیٹل پروڈکٹس بنانے ہوں۔

## نسخ: واضح اور عالمگیر

### تاریخی پس منظر

نسخ ("کاپی کرنا" یا "نقل کرنا") دسویں صدی میں بغداد میں ابھرا۔ مشہور خطاط **ابن مُقلہ** (وفات 940) نے اسے معیاری بنایا، اور بعد میں **ابن البواب** (وفات 1022) نے اسے کمال تک پہنچایا۔

نسخ کو قرآن کریم کی نقل کے لیے تیار کیا گیا تھا — اس لیے وضاحت اور پڑھنے میں آسانی اس کی بنیادی خصوصیات ہیں۔

### بصری خصوصیات

```
نسخ کی خصوصیات:
┌─────────────────────────────────────┐
│  • افقی بنیادی لکیر                  │
│  • کھڑے، سیدھے حروف                 │
│  • یکساں سٹروک کی چوڑائی             │
│  • کم لائن ہائٹ درکار               │
│  • آسان ڈیجیٹل رینڈرنگ              │
└─────────────────────────────────────┘
```

### استعمالات

نسخ استعمال ہوتا ہے:
- **قرآنی متون**: عالمی سطح پر معیاری رسم الخط
- **عربی اخبارات اور کتابیں**: مصر سے مراکش تک
- **ڈیجیٹل انٹرفیس**: زیادہ تر عربی ویب سائٹس اور ایپس
- **تعلیمی مواد**: نئے سیکھنے والوں کے لیے آسان

### ڈیجیٹل مثال

```css
/* نسخ فونٹ اسٹیک */
.arabic-naskh {
  font-family: 'Noto Naskh Arabic', 'Amiri',
               'Traditional Arabic', serif;
  line-height: 1.8;
  direction: rtl;
}
```

## نستعلیق: خوبصورت اور اظہاری

### تاریخی پس منظر

نستعلیق چودھویں-پندرھویں صدی میں ایران میں ابھرا۔ **میر علی تبریزی** کو اس کا موجد مانا جاتا ہے۔ نام "نسخ" اور "تعلیق" کا مرکب ہے — "لٹکتا ہوا نسخ"۔

یہ رسم الخط فارسی شاعری کے لیے تیار کیا گیا اور جلد ہی جنوبی ایشیا میں اردو کا معیاری رسم الخط بن گیا۔

### بصری خصوصیات

```
نستعلیق کی خصوصیات:
┌─────────────────────────────────────┐
│  • ترچھی بنیادی لکیر (اوپر سے نیچے) │
│  • جھرنے کی طرح بہتے حروف            │
│  • حروف کی عمودی تہہ بندی            │
│  • سٹروک کی چوڑائی میں تنوع          │
│  • بڑی لائن ہائٹ درکار              │
│  • پیچیدہ ڈیجیٹل رینڈرنگ            │
└─────────────────────────────────────┘
```

### استعمالات

نستعلیق استعمال ہوتا ہے:
- **اردو کے تمام متون**: پاکستان میں معیار
- **فارسی شاعری**: ایران اور افغانستان میں
- **خطاطی فن**: جمالیاتی کاموں میں
- **اخباری سرخیاں**: اردو میڈیا میں

### ڈیجیٹل مثال

```css
/* نستعلیق فونٹ اسٹیک */
.urdu-nastaliq {
  font-family: 'Jameel Noori Nastaleeq',
               'Noto Nastaliq Urdu',
               'Urdu Typesetting', serif;
  line-height: 2.4; /* بڑی لائن ہائٹ */
  direction: rtl;
}
```

## بصری موازنہ

### ایک ہی لفظ دو اندازوں میں

```
لفظ "پاکستان":

نسخ:     پاکستان   (افقی، سیدھا)
نستعلیق: پاکستان   (ترچھا، بہتا ہوا)
```

### جگہ کا استعمال

```
نسخ:
────────────────────────
یہ ایک لائن کا متن ہے جو افقی بہتا ہے
────────────────────────

نستعلیق:
╲
 ╲  یہ ایک لائن کا متن
  ╲  ہے جو ترچھا بہتا ہے
   ╲
```

## تکنیکی موازنہ

### گلفس کی پیچیدگی

| پہلو | نسخ | نستعلیق |
|------|-----|---------|
| بنیادی گلفس | ~200 | ~500+ |
| جوڑ (ligatures) | ~400 | ~20,000+ |
| فونٹ سائز | 100-500 KB | 5-15 MB |
| رینڈرنگ وقت | تیز | سست |

### لائن ہائٹ کا فرق

```css
/* نسخ - کم لائن ہائٹ کافی */
.naskh-text {
  line-height: 1.6;
  font-size: 16px;
  /* کل عمودی جگہ: ~26px */
}

/* نستعلیق - زیادہ لائن ہائٹ درکار */
.nastaliq-text {
  line-height: 2.2;
  font-size: 16px;
  /* کل عمودی جگہ: ~35px */
}
```

## کب کون سا استعمال کریں

### نسخ استعمال کریں

- عربی زبان کے پروجیکٹس
- ویب ایپلیکیشنز جہاں کارکردگی اہم ہو
- طویل متن (مضامین، کتابیں)
- موبائل ایپس
- ای میل اور سسٹم UI

### نستعلیق استعمال کریں

- اردو زبان کے پروجیکٹس
- پاکستانی مارکیٹ کی ایپس
- اخباری سرخیاں اور عنوانات
- شاعری اور ادبی متن
- برانڈنگ اور لوگو

## ڈیجیٹل چیلنجز

### نسخ - آسان نفاذ

```html
<!-- نسخ آسانی سے کام کرتا ہے -->
<p lang="ar" dir="rtl" class="naskh">
  مرحباً بالعالم
</p>
```

### نستعلیق - خاص توجہ درکار

```html
<!-- نستعلیق کے لیے اضافی غور -->
<p lang="ur" dir="rtl" class="nastaliq">
  خوش آمدید
</p>

<style>
.nastaliq {
  /* بڑی لائن ہائٹ */
  line-height: 2.4;

  /* متن کے نیچے اضافی جگہ */
  padding-bottom: 0.5em;

  /* اوور فلو نظر آئے */
  overflow: visible;
}
</style>
```

### فال بیک حکمت عملی

```css
/* نستعلیق فونٹ نہ ہو تو نسخ استعمال ہو */
.urdu-text {
  font-family: 'Jameel Noori Nastaleeq',
               'Noto Nastaliq Urdu',
               /* فال بیک نسخ فونٹس */
               'Noto Naskh Arabic',
               'Arial Unicode MS',
               sans-serif;
}
```

## کارکردگی کے نکات

### فونٹ لوڈنگ

```html
<!-- نسخ - چھوٹا فونٹ -->
<link href="noto-naskh.woff2" rel="stylesheet">
<!-- ~100KB -->

<!-- نستعلیق - بڑا فونٹ، preload ضروری -->
<link rel="preload" href="jameel-nastaleeq.woff2"
      as="font" type="font/woff2" crossorigin>
<!-- ~10MB -->
```

### سب سیٹنگ

```css
/* صرف ضروری حروف شامل کریں */
@font-face {
  font-family: 'Nastaliq Subset';
  src: url('nastaliq-subset.woff2');
  unicode-range: U+0600-06FF, U+FB50-FDFF;
}
```

## اہم نکات

۱. **زبان سے ملائیں**: عربی کے لیے نسخ، اردو کے لیے نستعلیق — یہ ثقافتی توقع ہے۔

۲. **کارکردگی کا خیال رکھیں**: نستعلیق فونٹس بڑے ہیں، لوڈنگ کی حکمت عملی بنائیں۔

۳. **لائن ہائٹ ایڈجسٹ کریں**: نستعلیق کو نسخ سے 30-50% زیادہ عمودی جگہ چاہیے۔

۴. **فال بیک فراہم کریں**: نستعلیق نہ ہو تو نسخ دکھائیں بجائے ٹوٹے حروف کے۔

۵. **حقیقی صارفین سے پوچھیں**: مقامی قارئین کی رائے سب سے قیمتی ہے۔
