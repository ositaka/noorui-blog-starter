## مقدمة

إذا عملت مع لغات بالخط العربي، فقد صادفت على الأرجح مصطلحين يظهران باستمرار: **النسخ** و **النستعليق**. يمثل هذان نهجين مختلفين جذرياً لكتابة الخط العربي، لكل منهما تاريخه وجمالياته وأهميته الثقافية.

للمطورين والمصممين، فهم هذا التمييز أمر حاسم. اختيار الأسلوب الخاطئ يمكن أن يجعل واجهتك تبدو "غريبة" للمتحدثين الأصليين — مثل استخدام خط Fraktur الألماني لموقع إنجليزي حديث.

يستكشف هذا الدليل كلا الأسلوبين بعمق: أصولهما، خصائصهما البصرية، تحدياتهما التقنية، وحالات الاستخدام المناسبة.

## النسخ: المعيار العالمي

### ما هو النسخ؟

النسخ (نسخ، بمعنى "النسخ" أو "الاستنساخ") هو الأسلوب الأكثر استخداماً للخط العربي. عندما ترى نصاً عربياً في كتاب أو صحيفة أو موقع إلكتروني، فهو على الأرجح مكتوب بخط على طراز النسخ.

### التطور التاريخي

ظهر النسخ في القرن العاشر تحت تنظيم **ابن مقلة** (٨٨٦-٩٤٠ م)، وزير وخطاط بارع في البلاط العباسي ببغداد. طور نظاماً تناسبياً حيث ترتبط جميع الحروف رياضياً بـ:

١. **النقطة المعينية** (نقطة): تُنشأ بضغط القلم بزاوية ٤٥°
٢. **الألف**: خط عمودي يحدد ارتفاعه نسب الحروف الأخرى

### الخصائص البصرية

للنسخ عدة خصائص مميزة:

١. **خط قاعدة أفقي**: الحروف تستقر على خط أفقي ثابت
٢. **انحناءات معتدلة**: أشكال مستديرة لكن ليست متطرفة
٣. **أشكال مدمجة**: الحروف تتراكم بكفاءة
٤. **وضوح القراءة**: النظام التناسبي يضمن تميز الحروف

### أين يُستخدم النسخ

- **العربية** (جميع البلدان)
- **الفارسية** (جنباً إلى جنب مع النستعليق في السياقات الرسمية)
- **الأردية** (للنصوص الدينية وبعض السياقات الرقمية)
- **الكردية**، **البشتونية**، **الأويغورية** ومعظم لغات الخط العربي الأخرى

## النستعليق: الخط الفني

### ما هو النستعليق؟

النستعليق (نستعلیق، مركب من "نسخ" + "تعليق") أسلوب خطي تطور في فارس. يتميز بتدفقه القطري المثير، حيث تبدو الحروف معلقة من خط مائل غير مرئي.

### التطور التاريخي

ظهر النستعليق في إيران في القرنين الرابع عشر والخامس عشر، وينسب تقليدياً إلى **مير علي تبريزي** (توفي ١٤٤٦). تقول الأسطورة أنه حلم بإوز يطير، وشكّلت تشكيلاتها الانسياب القطري المميز للخط.

### الخصائص البصرية

النستعليق ملفت بصرياً:

١. **خط قاعدة قطري**: يميل بشكل مثير من أعلى اليمين إلى أسفل اليسار
٢. **حروف معلقة**: تبدو الحروف معلقة من خط القاعدة
٣. **تباين شديد**: تنوع مثير بين الخطوط السميكة والرفيعة
٤. **تراكم عمودي**: تتراكم الحروف فوق بعضها بشكل متكرر
٥. **امتدادات عميقة**: الحروف ذات الذيول تنزل بعيداً تحت خط القاعدة

### أين يُستخدم النستعليق

- **الأردية** (الخط الأساسي والمفضل)
- **البنجابية** (خط شاهمخي في باكستان)
- **الكشميرية** (في كشمير تحت الإدارة الباكستانية)
- **الفارسية** (للشعر والتطبيقات الفنية)

## مقارنة جنباً إلى جنب

| الجانب | النسخ | النستعليق |
|--------|-------|----------|
| خط القاعدة | أفقي | قطري (٤٥-٦٠°) |
| تراكم الحروف | محدود | واسع |
| تباين الخطوط | معتدل | عالي |
| عمق الامتدادات | معتدل | عميق |
| عدد الرموز | ~٥٠٠ | ~٢٠,٠٠٠+ |
| ارتفاع السطر المطلوب | ١.٥-١.٨ | ٢.٠-٢.٥+ |
| النضج الرقمي | ممتاز | يتحسن |
| اللغات الرئيسية | العربية، الفارسية | الأردية، البنجابية |

## الآثار التقنية

### اختيار الخط

```css
/* للمحتوى العربي - النسخ */
.arabic-text {
  font-family: 'Noto Naskh Arabic', 'Traditional Arabic', serif;
  line-height: 1.8;
}

/* للمحتوى الأردي - النستعليق */
.urdu-text {
  font-family: 'Noto Nastaliq Urdu', 'Jameel Noori Nastaleeq', serif;
  line-height: 2.4; /* النستعليق يحتاج مساحة عمودية أكبر */
}
```

### اعتبارات ارتفاع السطر

التراكم العمودي والامتدادات العميقة للنستعليق تتطلب ارتفاع سطر أكبر بشكل ملحوظ:

```css
/* النسخ */
.naskh-paragraph {
  line-height: 1.6;
  margin-block: 1em;
}

/* النستعليق */
.nastaliq-paragraph {
  line-height: 2.2;
  margin-block: 1.5em;
  /* منع تداخل الامتدادات والصاعدات */
}
```

## الاعتبارات الثقافية

### توقعات القراء

استخدام أسلوب الخط الخاطئ يخلق تنافراً معرفياً:

- **القراء العرب**: يتوقعون النسخ. النستعليق يبدو أجنبياً وزخرفياً.
- **القراء الأردو**: يفضلون بشدة النستعليق. النسخ يشعرهم بـ"قراءة العربية" ويفقد هوية الأردية.
- **القراء الفرس**: يقبلون كليهما، لكن يربطون النستعليق بالشعر والنسخ بالنص اليومي.

## النقاط الرئيسية

١. **النسخ عالمي**: يعمل للعربية ومعظم لغات الخط العربي، مع دعم رقمي ممتاز.

٢. **النستعليق ضروري ثقافياً**: للمتحدثين بالأردية، النستعليق ليس مجرد تفضيل — إنه هوية.

٣. **التعقيد التقني يختلف بشكل كبير**: النستعليق يتطلب ٤٠ ضعف الرموز ومنطق تموضع ثنائي الأبعاد.

٤. **ارتفاع السطر مهم**: النستعليق يحتاج على الأقل ٢.٠-٢.٤ ارتفاع سطر لمنع التداخل.

٥. **اعرف جمهورك**: استخدام الأسلوب الخاطئ ينفر القراء فوراً.
